В папке Coder находится код с шифратором 
Как пользоваться :
 1.text.txt переходите
 2.В первую строчку пишем весь исходный текст (ЗАМЕЧАНИЕ: ИМЕННО В ПЕРВУЮ СТРОЧКУ БЕЗ ПЕРЕНОСА СТРОКИ)
 3.Во вторую строчку пишем первый пароль для ключа 
 4.В третью пишем 2 пароль
 5.В четвертую пишем 3 пароль
 6.В коде последняя переменная file_out , обязательно ее нужно изменить на путь к файлу (Decoder -> out.txt)
 В этот текстовый документ будет помещен результат шифровки 	
 В текстовых документах есть пример
В папке Decoder находится расшифратор 
Как им ползоваться :
 1.В документ Decoder -> passwords.txt нужно записать пароли , использующиеся при шифровке (1 пароль на каждую строчку)
 2.В finish.txt будет выведен результат
